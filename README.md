## Auro - Frontend part in React

Az Auro célja egy filmrajongók alkotta közösség formálása, ahol a felhasználók szabadon nyílváníthatnak véleményt az általuk értékelt darabokra.

## A weboldal az alábbi aloldalakat tartalmazza:
# - Filmek listája:
A felhasználó 100 (?) film közül választhat ún. kártyakomponensek közül, majd az adott filmet kiválasztva hozzászólhat.
# - Film "Tinder":
A 100 film közül egy random elemet feldob a rendszer, amelyre a felhasználó eldöntheti, hogy az adott film tetszik-e neki, vagy nem.
# - Regisztráció + Belépés:
A felhasználó létrehozhat saját fiókot, és amennyiben regisztrálta magát, be tud lépni abba.
# - Felhasználói fiók:
Sikeres regisztráció után a felhasználó láthatja a hozzászólásait és kedvenc filmjeit/sorozatait a saját fiókjában.

## Funkciók
# - Globális (az egész oldalt átfedő) sötét/világos mód:
Világos mód: Világos: fehér (#FFFFFF), halványszürke (#EDEDED), arany szöveg (#FFD60A)

Sötét mód: mélyszürke (#1C1C1E), lila akcent (#BF5AF2), világosszürke szöveg (#E5E5E7)

Betűtípus: SF Pro / Inter / Helvetica Neue
# - Reszponzív design:
Mobilbarát megjelenítés, azaz nem csak széles képernyőn jelenik meg igényesen a weboldal.
